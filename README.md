# Bank-Management-System
A simple console-based bank management system using JDBC and MySQL.

## Features
- Create, Read, Update, Delete (CRUD) accounts
- Console menu system
- JDBC with MySQL

## Requirements
- Java 17+ or 21
- MySQL server
- MySQL Connector/J
